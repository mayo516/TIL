<h3> 배열 인덱스 </H3>

0부터 시작한다. 

```
let colors = [ 'red' , 'orange' , 'yellow' ];

colors.length //3

colors[0] // 'red'
colors[1] // 'orange'
colors[2] // 'yellow'
colors[3] // 'undefined'

```

<h3> 배열 수정하기 </H3>

```
let colors = [ 'red' , 'orange' , 'yellow' ];

colors[3] // 'undefined'
colors[3] = 'blue'; // [ 'red' , 'orange' , 'yellow', 'blue']

```
